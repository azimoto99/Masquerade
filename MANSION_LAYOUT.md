# Mansion Layout & Visual Design

## Overall Mansion Architecture

### Mansion Overview
The mansion is a sprawling Victorian Gothic estate with 12 interconnected rooms arranged in a logical floor plan. The design emphasizes atmosphere, navigation challenges, and strategic positioning for gameplay.

### Floor Plan Layout
```
[Attic] ← → [Master Bedroom]
   ↑           ↑
[Chapel] ← → [Gallery] ← → [Conservatory]
   ↑           ↑           ↑
[Wine Cellar] ← → [Billiard Room] ← → [Study]
   ↑           ↑           ↑
[Kitchen] ← → [Dining Hall] ← → [Library]
                   ↑
              [Grand Ballroom]
```

### Room Connectivity
- **Main Corridors**: Grand Ballroom connects to all major rooms
- **Secret Passages**: Hidden routes between select rooms
- **Door States**: Can be locked, ajar, or barricaded
- **Line of Sight**: Limited by walls, furniture, and room geometry

## Visual Style Guide

### Color Palette
**Primary Colors**:
- Deep burgundy (#722F37)
- Rich gold (#D4AF37)
- Midnight blue (#191970)
- Antique white (#FAEBD7)
- Charcoal gray (#36454F)

**Accent Colors**:
- Blood red (#8B0000) for scares
- Ghostly white (#F8F8FF) for apparitions
- Bronze (#CD7F32) for metallic elements
- Forest green (#228B22) for foliage

### Lighting System
**Dynamic Lighting States**:
- **Normal**: Warm candlelight (orange tones)
- **Haunted**: Flickering blue-white (cold, eerie)
- **Dark**: Minimal red emergency lighting
- **Emergency**: Bright white floodlights

**Lighting Effects**:
- Shadows that move and distort
- Light sources that can be extinguished
- Particles (dust motes, wisps)
- Lens flare on glass surfaces

### Art Style
**Overall Aesthetic**: Detailed 2.5D isometric view
- **Line Art**: Clean, gothic architectural lines
- **Textures**: Rich, hand-painted style materials
- **Animation**: Subtle movement (candles flickering, curtains swaying)
- **Effects**: Particle systems for dust, mist, and ectoplasmic energy

### Asset Specifications
**Room Backgrounds**: 512x384 pixel sprites
**Furniture**: 64x64 to 128x128 pixel sprites
**Characters**: 32x48 pixel sprites with 8-directional animation
**Effects**: 256x256 pixel particle systems
**UI Elements**: Vector graphics, scalable

## Camera and Perspective

### Isometric View
- **Angle**: 30-degree isometric projection
- **Zoom Levels**: 0.5x to 2x for different room sizes
- **Depth**: Layered rendering for foreground/background
- **Parallax**: Subtle background movement

### Room Transitions
- **Door Animations**: Smooth opening/closing
- **Fade Effects**: Brief screen fade during room changes
- **Loading States**: Seamless transitions with room previews

## Accessibility in Visual Design

### Visual Clarity
- **High Contrast**: Important elements stand out
- **Color Coding**: Different room types have distinct color schemes
- **Icon Consistency**: Universal symbols for interactive elements

### Motion Sensitivity
- **Reduced Motion Option**: Toggle for animation intensity
- **Warning System**: Pre-notification for intense visual effects
- **Alternative Indicators**: Non-motion cues for events

This mansion layout provides a rich, atmospheric environment with strategic depth while maintaining visual clarity and gameplay balance. Each room serves specific narrative and mechanical purposes while contributing to the overall Gothic mystery theme.
